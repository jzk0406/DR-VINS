## 1. 项目简介

DR-VINS 是一个面向退化视觉场景的视觉惯性 SLAM 实验系统。项目基于 VINS 系列视觉惯性估计框架，引入深度特征提取与匹配前端，用于提升系统在低纹理、光照变化、强光过曝和室内外切换等困难场景下的定位稳定性和鲁棒性。
